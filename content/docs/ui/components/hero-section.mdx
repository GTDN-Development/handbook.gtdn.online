  HeroBackground,

    <HeroContent>
        <HeroTitle>Hero Title</HeroTitle>

<Steps>
<Step>
**Install the Container component**


This component depends on the [Container](/docs/ui/components/container) component. Make sure you have it installed first.
</Step>

<Step>
**Install dependencies**

```bash
npm install @radix-ui/react-slot
```
</Step>

<Step>
**Create the component file**

```bash
touch src/components/ui/hero.tsx
```
</Step>
