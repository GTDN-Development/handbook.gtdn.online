## Usage

Add the component to your root layout:

```tsx
import { TailwindScreen } from "@/components/ui/tailwind-screen";

export default function RootLayout({ children }) {
  return (
    <html>
      <body>
        {children}
        <TailwindScreen />
      </body>
    </html>
  );
}
```

The component automatically hides in production and displays as a fixed element in the bottom-left corner during development.
