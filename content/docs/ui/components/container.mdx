##  Usage

Use the `asChild` prop to render the Container as a different element while preserving its styling:

```tsx
<Container asChild>
  <section>
    <h1>Section with container constraints</h1>
  </section>
</Container>
```
This is useful when you need semantic HTML elements or want to avoid unnecessary div nesting.
