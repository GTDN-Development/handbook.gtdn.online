## Core Tools

- **Google Workspace** → Email, Calendar, Drive, Docs, Sheets, and especially **Google Chat/Meet** for quick questions, minor updates, meetings, and demos. Your `@gtdn.online` account is the hub for everything.
- **ClickUp** → Project management, task tracking, time tracking, and documentation. Where work gets organized and tracked.
- **Figma** → Design files, prototypes, and handoffs. Designers and developers collaborate here.
- **GitHub** → Code hosting, version control, code reviews, and CI/CD. All repositories live here.

## Access

You'll get access to all tools during onboarding.
